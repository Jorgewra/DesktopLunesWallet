# Wallet Lunes Desktop Offeline

Sending and receiving coins LUNES Basic

## Getting Started

a. Seed Creation and Importation ['ok']

b. Send and receive ['ok']


### Prerequisites

1 - JAVA 8 JRE

2 - Maven

### Installing

```
mvn clean install
```
